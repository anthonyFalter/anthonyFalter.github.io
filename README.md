# Ross Elopre Portfolio
This is my personal portfolio website designed to host whatever I'm currently focused on, goals I have set and stories I have to share.

## Project Structure

- index.html — the main homepage
- style.css — shared styling for the site
- script.js — theme toggle and small interactive behavior
- assets/ — images and other static assets

## Blog Documentation

The site now supports a blog section with:

- a main blog listing page
- individual blog article pages
- the same visual style as the rest of the portfolio

### Adding a new blog

1. Create a new HTML file for the blog article, for example: blog/my-first-post.html
2. Add a link to it from the main blog listing page in blog/index.html
3. Include the following information for the post:
   - title
   - short description
   - published date
   - full article content

### Editing an existing blog

1. Open the blog article HTML file you want to update
2. Replace the content inside the article section
3. Keep the page structure consistent with the other blog posts

### Blog page structure

Each blog article page should include:

- a back link to return to the blog list
- the published date
- the blog title
- the full article content

### Example blog list item

Each item in the blog listing should include:

- title
- short description
- published date
- a link to the full article page
